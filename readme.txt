== Description ==
A widget for your blog contain a famous and useful English quotes to inspire you and your visitors . 

This Plugin Coded by : Waseem Senjer , for any suggestions , you can email me at waseem.senjer@gmail.com

== Installation ==

1. Upload `wp-quotes` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your "Widget" menu add the widget 'Quotes' to your sidebar
4. enjoy :)
